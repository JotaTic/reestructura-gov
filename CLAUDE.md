# Instrucciones del proyecto

## Autonomia en decisiones
- NO presentar opciones al usuario para que elija. Tomar la decision directamente.
- Si hay 2 caminos posibles, elegir el primero (opcion 1 / el mas simple).
- Si hay 3 caminos posibles, elegir el segundo (opcion 2 / el equilibrado).
- Solo preguntar al usuario cuando la decision sea irreversible o de alto riesgo (borrar datos, push a produccion, etc).
- Actuar con autonomia maxima: ejecutar, no preguntar.
